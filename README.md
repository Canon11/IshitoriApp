# IshitoriGame
GUIで操作する1vs1の石とりゲーム

## Description
石取りゲーム。２人プレイヤーで行い、交互にプレイをする。最後の1つの石をとったプレイヤーが負けであり、1度に隣り合った最大3つの石を取れる。
GUIウィンドウ上で動く。

***DEMO:***

![circleanimationmuvie](https://github.com/Canon11/IshitoriGame/blob/master/gif/Demo.gif)

## Features
- あえてGUI部品で作ることで、シンプルかつ動作を軽くしてミニゲーム感が出ている（多分）。
- 取れる石が赤色でハイライトされる。
- リスタートで何度もプレイが可能。

## Requirement
- java 1.7

## Usage
```
$ cd IshitoriGame
$ javac *.java
$ java MyApplication
```
