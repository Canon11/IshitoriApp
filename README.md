# IshitoriApp
GUIで操作する1vs1の石とりゲーム

## Description
石取りゲーム。２人プレイヤーで行い、交互にプレイをする。最後の1つの石をとったプレイヤーが負けであり、1度に隣り合った最大3つの石を取れる。
GUIウィンドウ上で動く。

***DEMO:***

![Demo](https://image-url.gif)

## Features
- あえてGUI部品で作ることで、シンプルかつ動作を軽くしてミニゲーム感を出している。

## Requirement
- java 1.7

## Usage
```
$ cd IshitoriApp
$ javac *.java
$ java MyApplication
```
